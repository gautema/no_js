<details> - Accordion uten JS

Hva er HTML?

HTML (HyperText Markup Language) er grunnlaget for alle nettsider. Det strukturerer innhold og gir mening til tekst, bilder og andre elementer.

Hva er CSS?

CSS (Cascading Style Sheets) styrer utseendet til nettsider. Det kontrollerer farger, layout, typografi og animasjoner.

Trenger jeg JavaScript?

Ikke alltid! Som denne presentasjonen viser, kan du gjøre mye med bare HTML og CSS. JavaScript er kraftig, men ikke alltid nødvendig.
