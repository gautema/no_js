CSS Anchor Positioning

Posisjon elementer relativt til hverandre - uten JavaScript!

Profil Innstillinger Hjelp
Logg ut 
/* Anchor the button */
.dropdown-trigger {
  anchor-name: --menu;
}

/* Position menu below button */
.dropdown-menu {
  position: fixed;
  position-anchor: --menu;
  top: anchor(bottom);
  left: anchor(left);
  margin-top: 8px;
}
